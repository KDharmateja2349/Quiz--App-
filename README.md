# 🎯 Simple Quiz App

A responsive and interactive Quiz Application built using **HTML, CSS, and JavaScript**. Users can answer multiple-choice questions, receive instant feedback, and view their final score at the end of the quiz.

## 🚀 Features

* Multiple-choice quiz questions
* Instant answer validation
* Highlights correct and incorrect answers
* Score tracking system
* Play Again functionality
* Responsive and clean user interface
* Pure JavaScript (No external libraries)

## 🎮 How to Use

1. Open the application in your browser.
2. Read the question carefully.
3. Select the answer you think is correct.
4. Correct answers will be highlighted in green.
5. Incorrect answers will be highlighted in red.
6. Click the **Next** button to proceed.
7. At the end of the quiz, your total score will be displayed.
8. Click **Play Again** to restart the quiz.
